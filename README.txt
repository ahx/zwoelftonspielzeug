Zwölftonspielzeug 
------------------
Frei nach den Erkenntnissen von 
Joseph Matthias Hauer, dem Erfinder der Zwölftonmusik.

Von Andreas Haller <andreashaller@gmail.com>

A \"Zwölftonspiel\" (twelve-tone-game) is a twelve-tone-technique by Josef Matthias Hauer.

Joseph Matthias Hauer is the original creator of the twelve-tone technique (dodecaphony).

Install
* `sudo gem install bundler`
* `cd zwoelftonspielzeug`
* `bundle install`

Usage
* Run ./start
* Make shure pd/start.pd is running
* Open index.html in Google Chrome
