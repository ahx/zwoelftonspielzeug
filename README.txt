Zwölftonspielzeug 
=================
Frei nach den Erkenntnissen von Josef Matthias Hauer, dem Entdecker der Zwölftonmusik.

Für die Installation des Programms sind zur Zeit leider ein paar Schritte notwendig.

Installation (Mac OSX 10.6)
---------------------------
 * PureData installieren        (http://puredata.info/)
 * XCode installieren           (http://developer.apple.com/mac/)
 Im Terminal:
 * sudo gem update --system      (udpate auf Rubygems 1.3.6)
 * sudo gem install bundler     (installiert Paketmanager für Ruby)
 * cd zwoelftonspielzeug    
 * bundle install 

Start
---------------------------  
  * Das "start" script starten
  * Sobald das PD patch (pd/start.pd) läuft, mit [ENTER] das Spiel im Terminal starten
  * index.html in Google Chrome öffnen (http://www.google.com/chrome)
  * Ist das Akai LPD8 an PD angeschossen? (vielleicht klappts auch mit einem anderen controller)
  * Alternativ zum MIDI Controller, kann die Anwendung auch über die index.html gesteuert werden.
